# Packet Sniffer

A Python-based packet sniffer built using Scapy for network traffic analysis.

## Features

- Captures live network packets
- Displays source IP addresses
- Displays destination IP addresses
- Detects TCP, UDP, and ICMP traffic
- Tracks protocol statistics
- Displays packet length information

## Technologies Used

- Python 3
- Scapy
- Kali Linux

## Installation

```bash
pip install scapy
```

## Usage

```bash
sudo python3 packet_sniffer.py
```

## Example Output

```text
===== Packet Captured =====
Source IP: 192.168.10.17
Destination IP: 13.107.226.43
Protocol: TCP
Packet Length: 66 bytes
```
